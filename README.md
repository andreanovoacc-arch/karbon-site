# KARBÓN — Carnicería Premium

Sitio web de KARBÓN, carnicería premium al carbón en San Salvador.

Todo el sitio vive en un solo archivo (index.html): catálogo, carrito y checkout con factura digital, historial de pedidos, Club Karbón, planificador de menú semanal y recetas — en español e inglés.

Para verlo localmente: abre index.html en cualquier navegador. No necesita servidor ni instalación.
